Keystroke Authentication

Behavioral biometric authentication powered by a Transformer encoder

How it works: Select a subject ID, then type the password below. The system captures your keystroke timing patterns and authenticates you using a trained Behavioral Transformer Encoder (BTE). Type naturally — do not pause mid-password.

Type the password exactly as shown above. Press Enter or click Authenticate when done.
